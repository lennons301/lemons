LLM-assisted meal generation (v1 search-based suggestions, v2 Claude API) is a fundamental goal of the app, not a polish item.

**Why:** The user considers this a core aiming point of the platform, not a nice-to-have.

**How to apply:** When discussing build order or phases, treat meal generation as its own dedicated phase. Don't lump it with polish/misc work. It deserves its own design doc and implementation plan.
